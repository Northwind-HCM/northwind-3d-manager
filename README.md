# Northwind 3D Manager

Eigenständige Next.js-App für Filamentverwaltung und Druckkostenkalkulation.

## Start in Visual Studio Code

1. Projektordner in VS Code öffnen
2. Terminal öffnen
3. `npm install`
4. `npm run dev`
5. Im Browser `http://localhost:3000` öffnen

## Enthalten

- modernes Dashboard
- Filamentlager mit Suche und Bestandsampel
- Bestand per Plus/Minus ändern
- lokales Speichern im Browser
- Kosten- und Etsy-Verkaufspreisrechner

## Nächste Ausbaustufe

Firebase, Produktverwaltung, Produktionsaufträge, Druckerverwaltung und Excel-Import.
